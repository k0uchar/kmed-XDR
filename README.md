# KMED XDR
**AI-Powered Autonomous SOC Platform**

## Project Vision
KMED XDR is a free MVP for PFE 2025. It collects logs, analyzes alerts with AI, and generates reports. Designed to be expandable for companies or personal SaaS.

## Structure
- backend: Python + FastAPI
- frontend: HTML + Tailwind dashboard
- agents: log collection scripts
- workflows: N8N automation flows
- docs: architecture diagrams, weekly notes
